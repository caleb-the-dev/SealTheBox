Key Questions — The Big Picture

Q1. Overall: does mid-match tab mutation feel like strategic depth (I need to plan around it) or chaos (it just happens to me)?

Q2. Which 2–3 boxes felt best? What made them work?

Q3. Which boxes felt like noise — barely noticeable, or just frustrating with no interesting decisions?

Q4. Is Mitosis fun or frustrating? (This is the litmus test for high-cognitive-load boxes.)

Q5. Did Fading Decoys' phantom tabs feel fair on first discovery, or like a cheap surprise?

Q6. Any boxes you'd cut immediately? Any you'd want to buff/nerf before merging?

How to Test

  • Reach BHV boxes quickly: Press T (or DEV button) → "Win Current Match" repeatedly to advance to medium/hard/boss tiers where BHV boxes appear.
  • Or use "Win Entire Series" in the dev menu to jump to act 2/3 matches where harder boxes show up.
  • Watch for the status bar messages — each BHV hook prints a message when it fires (e.g. "Rising Tide — all tabs +1").
  • Fading Decoys: decoy tabs (3, 5, 7 phantom values) look identical to real tabs on R1–2. Vanish at R3 start.
  • Mitosis: sealing a 12 immediately adds a 6; sealing the spawned 6 adds a 3; the 3 is sealable (value < 6) and doesn't spawn further.
  • Revenant: end a round without sealing any tab to trigger it — the lowest tab you already sealed comes back.
