0 / 3 tested
How to test: Press T (or DEV button) → Force Storm Box → / Force Cleanse Box → / Force Borrowed Time → to jump straight to each ENTRY box. You can also go via Go to Box → menu.

Exclamation badge [!]: Each ENTRY box shows a glowing [!] badge next to the box name at the top left. Hover it to see the tooltip describing the entry effect before you play. Make sure the tooltip appears and is readable.

Status message: When the match starts, the status bar at the bottom should show a message describing what just happened (e.g. "Storm Box — bonus d6 added to your pool!").

What you're testing: Do these boxes feel different from turn 1? The entry effect should shape your strategy immediately — before you ever roll a die. Ask: did the effect change how you approached the match?

KEY QUESTIONS THIS SLICE IS TESTING

  1. "Deep breath" beat? (Cleanse Box) — Refilling all charges feels like a mid-run reset. Does it create a moment of relief, or does it feel like a stat bump you forget about?
  2. Fair tradeoff or a tax? (Borrowed Time) — −1 HP for +1 round. Does the extra round feel like meaningful breathing room, or does it feel like the box is punishing you just for existing?
  3. Does the bonus die get noticed? (Storm Box) — An extra d4/d6/d8 added to your pool. Does it change what you can seal in a round, or does it get lost in the noise of 7→8 dice?
  4. Did you know the effect fired? — Across all 3 boxes: was the entry effect communicated clearly enough (badge tooltip + status message) that you knew what happened?
  5. Did it change your turn-1 plan? — Cleanse = go for a different ability. Storm = more options. Borrowed Time = slightly more HP-conservative. Did any of these actually change how you played?
