How to test: In-game, press T (or DEV button) → Go to Box → → pick the box name. Play 2–3 rounds to feel the modifier, then rate below.
What you're testing: Each box applies a rule to your dice every roll. The rule is printed in the Modifier column. Ask yourself: did I notice it? Did it change my decisions? Was it fun?

KEY QUESTIONS THIS SLICE IS TESTING

  1. Noticeable? Did the modifier visibly change what you rolled — or was it subtle to the point of invisible?
  2. Strategy-changing? Did the modifier make you think differently about which tabs to target?
  3. Fair? Did the modifier feel like a rule you could play around, or did it feel arbitrary / out of your hands?
  4. Difficulty calibrated? Medium boxes should feel like Cluster of Twos / Triple Triplets. Hard boxes should feel like High Wall / Exact Evens.
